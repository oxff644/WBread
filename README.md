## 前言

Wbread是一个简单的日志分析工具，设计初衷是为了方便安全人员日常工作中，或者应急响应时的日志分析工具。主要有以下亮点：

* 方便、快速、一键运行
* 界面整洁
* 精准度高


## 环境准备

1. python3.8
2. 基本类库
```plain
ipdb-python
termcolor==1.1.0
dataclasses==0.8
```

### 运行

```plain
python3.8 main.py
```
